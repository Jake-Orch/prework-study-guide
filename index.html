CSS

  • There are 3 forms of CSS; Inline CSS, Internal CSS and External CSS
  • Inline CSS is used to apply a unique style to a single HTML element
  • Internal CSS is when you add CSS to the head section of HTML documents
  • External CSS is CSS in a seperate file that can be accessed by creating a link within the head section of the HTML document using a "link"
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The selector points to the HTML element you want to style
  • The decloration block contains one or more declorations to be made to the selected elements

Git

  • Git is a version control system which allows for managing and tracking each version of your project
  • git commands ALWAYS start with git
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull: fetch and merge commits from the tracking remote branch to our local branch
  • git add -A: to commit all files in the working branch
  • git commit -m "commit desctiption": commits all files in working branch, the -m allows us to associate a message with our commit
